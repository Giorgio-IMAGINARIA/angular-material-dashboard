MATERIAL DASHBOARD
A dashboard made with Angular2 + Angular Material

1) - git clone https://github.com/Giorgio-IMAGINARIA/angular-material-dashboard.git
2) - type "npm install" in the root directory
4) - type "npm run start" in the root directory
5) - navigate to "http://localhost:8080/" in your browser
6) - enjoy :)
